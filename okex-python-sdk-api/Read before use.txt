
1. 调用我们的接口均需要科学上网：
rest可尝试Ping www.okex.com
ws可尝试Ping real.okex.com
检查是否可ping通，需客户端排除解决网络问题
服务器建议 香港阿里云

2. v3 api文档地址： https://www.okex.com/docs/zh/
申请v3API网址： https://www.okex.com/account/users/myApi

3. 关于K线数据的获取：
币币：
最多可获取最近2000条（例如一分钟为粒度可拿到最近2000/60=33.33小时的K线数据）
单次请求的最大数据量为200条

交割与永续：
最多可获取最近1440条（例如一分钟为粒度可拿到最近1440/60=24小时的K线数据）
交割单次请求的最大数据量为300条
永续单次请求的最大数据量为200条

4. 关于限速规则：
公共数据根据ip限速，若传入有效的个人信息即api key则根据userid限速
个人数据根据账号即userid限速
各接口相关限速规则以api文档为准
如有需求可申请多个子账号来进行分流

ws根据ip限速
连接限制：1次/s
订阅限制：每小时240次（按send次数计）

5. 关于高级限价委托：
Post only 为只做挂单，等待别人来和你成交，不会主动吃单
FOK 全部成交或立即取消，即以你委托的价格在当前深度中能否可以把你委托的数量全部成交，才会下单成功，否则会立即撤销，不会出现部分成交的情况
IOC 立即成交并取消剩余，即以你委托的价格进行成交，不管成交多少取消剩余未成交的订单，会出现部分成交的情况

6. 若使用ws中遇到问题可参考同级目录下 May be could help you.txt 文件
